# Employee Manager
Project deployed on Render, Firebase & CleverCloud database. <br>
https://primeraapp-c3a9a.web.app/ 
 <br> <br>
Technologies, Frameworks & Tools:
- Backend:
  - Java 17
  - SpringBoot
  - MySql
  - Postman

- Frontend:
  - Angular 17: HTML & TypeScript
  - Bootstrap
